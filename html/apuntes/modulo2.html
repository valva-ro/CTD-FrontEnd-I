Modelo de caja

Cada elemento en TML es una caja compuesta de mrgenes (margin), bordes (border), relleno (padding) y finalmente contenido (content). Con CSS podemos manipular todas estas propiedades para cambiar la apariencia de cada elemento. A su vez, estas propiedades pueden aplicarse de manera diferente a los cuatro lados de la caja (top, bottom, left, right).

Padding

Podemos asignarle 1, 2, 3 o 4 valores.

xxxxxxxxxx
padding: 12px;                    /* se aplica 12px a todos los lados */
padding: 12px, 20px;              /* 12px arriba y abajo, 20px izquierda y derecha */
padding: 12px, 20px, 16px;        /* 12px arriba, 20px izquierda y derecha, 16px abajo */
padding: 12px, 20px, 16px, 14px;  /* 12px arriba, 20px derecha, 16px abajo, 14px izquierda */

Border

Podemos asignarle el estilo de lnea, espesor y color:

  • Espesor: cualquier unidad de medida
  • Color: cualquier color en cualquier notacin vlida de CSS

Margin

Es el margen exterior del elemento, nos sirve para separar una caja de otra. Al igual que el padding podemos asigarnle entre 1 y 4 valores.

Box sizing

Por defecto el valor ser content-box y el comportamiento ser el de aplicarle el ancho y alto que definamos al conenido del elemento.

Si le asignamos el valor border-box el ancho y alto que indiquemos tomar en cuenta no solo el contenido del elemento, sino tambin el padding y el borde, dejando solo el margen por fuera.

Es muy comn asignarle box-sizing: border-box a todos los elementos del sitio.

Width y Height

De manera predeterminada el ancho (width) de un elemento ser el 100% de su padre contenedor, pero podemos asignarle un valor nosotros usando cualquier unidad de medida.

Por su parte la altura (height) predeterminada de un elemento ser la que su contenido le proporcione, ya sea de bloque o de lnea. Al igual que el ancho, podemos asignarle un valor usando cualquier unidad de medida. En general no se recomienda usa porcentajes.

Posicionamiento

  • relative: se desplaza el elemento desde su posicin original sin modificar el flujo original de los dems elementos.

  • absolute: se desplaza el elemento desde el body. Modifica el flujo original de los dems elementos, ya que al moverse ese espacio "queda vaco" y otros elementos lo ocupan.

  • relative + absolute: si nuestras cajas hijas estn dentro de otra caja padre, el punto de referencia seguir siendo el body a menos que el posicionamiento del padre del fuera relative.

  • fixed: desplaza el elemento tomando como referencia la ventana del navegador. El elemento siempre se mantendr fijo con respecto a los costados de la ventana.

  • sticky: es considerado un hbrido entre relativo y dijo ya que es tratado como un elemento relativamente asta que cruza un umbral especificado, en cuyo punto se trata como fijo asta que alcanza el lmite de su padre.
